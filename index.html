CAL Duty (Pro) — 內建離線說明
✈️ CAL Duty (Pro v6.8.3) 雙語使用手冊 / Bilingual User Guide
【中文版使用說明】
這是一款專為中華航空飛行員設計的「法定飛時與休時計算工具」，嚴格遵循 FOM 3.4 Crew Scheduling 規範。
⏱️ 快速上手：標準使用 5 步驟
這套系統的設計邏輯是 「由上到下、由左到右」，並區分 「計畫階段 (Plan)」 與 「執行階段 (Actual)」。
1. 初始基本設定
  • 選擇航班日期 (Flight Date)：位於頂部左側。此日期作為換算 UTC 與 Local Time 的基準。
  • 選擇派遣配置 (Crew)：點選 Single (單重)、Multi (多重) 或 Double (雙重)，這會決定 Max FDT 與休時基準。
  • 選擇時區模式：點擊 ALL UTC / ALL LOCAL 一鍵切換所有時間輸入格的預設時區。
2. 建立航段與計畫時間
  • 輸入機場代碼 (Code)：輸入 3 碼 IATA 或 4 碼 ICAO 代碼（如輸入 RJAA 會自動轉為 NRT）。
  • 點擊卡片下方的 + 新增航段，⤵ 新增轉降 (Divert)。
  • 輸入表定時間 (STD / STA)：輸入 4 位數字（如 0830），系統會自動換算時區。
  • iPad 浮動數字鍵盤：在 iPad 點按 HHMM 時間欄位時，會直接顯示 app 內建數字鍵盤（含 ⌫／清除／完成）；v6.8.3 延續 v6.8.2 的 iPad 浮動鍵盤修正，並把版本更新標註重構為卡片式雙語 Release Notes，現在更容易閱讀與檢索。
💡 提示：輸入完成後，底部 HUD 會自動算出您的報到時間 (Report) 與 最大任務時間 (Max FDT)。
3. 特殊條件與例外
  • 點擊頂部右上角 🔽 展開進階面板：
  • Ext FDT：M/D Crew 遇不可抗力，機長裁量權延長 2 小時。
  • Night：夜航自動扣減 2 小時。
  • No Bunk：M/D Crew 無床鋪，24h 飛時限縮為 12h。
  • PREV FT：檢查 24h Rolling，點擊輸入前趟 IN Z 與 FT。
  • JZ LBO：輸入聯管 ACARS 給予的最後合法後推時間。
  • LBO Warn（倒數預警）：以「目前時間」計算距離下一個尚未 OUT 航段的 LBO 剩餘分鐘（LBO Rem）。LBO Rem ≤ C 進入 CAUTION；≤ A 或超過 LBO 進入 ALERT。Pop 為進入警戒時彈出一次提示。
4. 執行階段與實際時間
  • 遇到延誤 (Report Lock 🔒)：航班延誤時，點擊底部 Report 旁的鎖頭，固定極限計算基準。
  • 填寫實際時間 (ACT OUT / IN / FT)：輸入實際後推與靠橋時間。
  • 機上輪休 (Rest Swap Alarm 💤)：(M/D Crew) 點擊紫色 💤 按鈕，輸入 Block Off 時間，系統自動算出輪換時間表。
5. 外站休息與 Next Duty
  • 外站休息 (Hotel / Split)：休息 >3h，開啟航段右上 HOTEL 開關，輸入 C/I 與 C/O 時間。
  • Next Duty Planner：輸入下趟目的地 (如 ANC) 或地面任務 (如 SIM)。系統自動檢查時差與 48h 豁免條件（如：同地、時差≤3h、地面任務皆豁免）。
🚦 狀態判讀與技巧
  • SAFE / CAUTION / ALERT：點擊頂部狀態條可看詳細診斷原因。
  • LBO (最後後推)：實際 OUT 晚於 LBO 即為超時 (紅線/橘線)。
  • ACM (Deadhead)：點擊兩站連線處的 Flight Leg 可切換為 ACM。
  • 儲存航線 (Routes 📂)：點擊頂部資料夾可儲存/匯出/匯入常用航線。

【English User Guide】
This is a legal flight duty and rest time calculation tool designed for China Airlines pilots, strictly compliant with FOM 3.4 Crew Scheduling regulations.
⏱️ Quick Start: 5 Steps
The system is designed "top-to-bottom, left-to-right", divided into "Plan" and "Actual" phases.
1. Initial Setup
  • Flight Date: Located at the top left. Used as the base date for UTC/Local Time conversions.
  • Crew: Select Single, Multi, or Double crew complement to determine Max FDT and rest limits.
  • Timezone Mode: Click ALL UTC / ALL LOCAL to toggle the default timezone for all input fields.
2. Planning
  • Code: Input 3-letter IATA or 4-letter ICAO codes. Tap + to add legs, ⤵ to add diverts.
  • STD / STA: Input 4 digits (e.g., 0830). The system automatically formats and converts timezones.
  • iPad floating number pad: On iPad, tapping an HHMM field opens the built-in number pad (with ⌫ / Clear / Done); v6.8.3 carries forward the v6.8.2 iPad floating-pad fix and restructures update annotations into a bilingual card-based Release Notes panel for cleaner reading and scanning.
💡 Tip: Bottom HUD will auto-calculate your Report Time and Max FDT once STD/STA are entered.
3. Exceptions
  • Click 🔽 top-right to expand advanced panel:
  • Ext FDT: PIC discretion extension (+2h) for M/D crew.
  • Night: Auto-detects 2100L-0700L departure, -2h FDP limit.
  • No Bunk: Limits 24h FT to 12h without bunk for M/D crew.
  • PREV FT: Input previous arrival/FT for 24h rolling check.
  • JZ LBO: Input JZ provided ACARS last block-out time.
  • LBO Warn (countdown): Uses the current time to compute minutes remaining to the LBO of the next not-yet-OUT leg (LBO Rem). LBO Rem ≤ C → CAUTION; ≤ A or beyond LBO → ALERT. Pop shows a one-time toast when entering warning states.
4. Execution
  • Report Lock (🔒): Tap the lock icon near Report time during delays to freeze the limit baseline.
  • ACT OUT / IN / FT: Input actual block-out/in times or known block time (FT).
  • Rest Swap Alarm (💤): (M/D Crew) Tap the purple 💤 chip and input Block Off to auto-calculate rest swap schedules.
5. Rest & Next Duty
  • Hotel / Split: For rest >3h, toggle HOTEL on the leg card and input C/I & C/O times.
  • Next Duty Planner: Input next destination or ground duty. Auto-checks TZ differences and 48h rule exemptions.
🚦 Status Indicators & Pro Tips
  • SAFE / CAUTION / ALERT: Green (Legal), Yellow (Approaching Limit), Red (Exceeded). Tap for details.
  • LBO: Actual OUT later than LBO indicates a violation.
  • ACM (Deadhead): Tap "Flight Leg" between stations to switch to ACM (Deadhead).
  • Routes (📂): Click the folder icon to save, export, or import routes.
Tip: 你可以用 📖 於頂部工具列隨時開啟本手冊（離線可用）。
